## 安裝

以下將會引導你如何安裝此專案到你的電腦上。

-   Node.js v20.9.0

### 取得專案

```
git clone git@github.com:ellaYang1227/node-todolist-api.git
```

### 移動到專案內

```
cd node-todolist-api
```

### 安裝套件

```
npm install
```

### 運行專案

```
npm run start
```

### 使用 nodemon 啟動伺服器
需先安裝 nodemon 套件
```
nodemon server.js
```

### 瀏覽器開啟專案

```
http://127.0.0.1:3005/
```

## 專案技術

- Node.js v20.9.0
- uuid v9.0.1
- Render 部署 API
- Postman API 測試工具
